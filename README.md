# ITCS4102 Final Project

## Comparing Python, Rust, and Go in the Age of Generative AI

This project compares Python, Rust, and Go based on readability, performance, memory safety, concurrency, and usefulness in modern software development.

## Project Goals

- Research the history and design of Python, Rust, and Go
- Compare syntax and readability
- Develop similar programs in each language
- Analyze performance and coding quality
- Discuss how Generative AI affects programming languages

## Tools Used

- Visual Studio Code
- GitHub
- Git
- Python
- Rust
- Go
- ChatGPT

## Programs Included

1. Prime Number Generator
2. API Request Program
3. Concurrent Task Executor

## Conclusion

Python is best for readability and AI-related development, Rust is strongest for performance and memory safety, and Go is useful for cloud-based and concurrent programming.
